CSSのボックスモデル

概要

この記事では、要素の表示される間隔を調整するためにCSSを使う方法について記します。

ボックスについて

ブラウザは要素を表示する際に間隔を取ります。間隔を取る箇所は4つあります。

まず、要素は中央に表示されます。その周りには、パディングが存在します。そしてその周りはボーダーです。さらにその周りのマージンにより、要素がその他と分離されます。

図表省略

パディング、ボーダー、およびマージンは、要素の上下左右それぞれに異なるサイズを指定することができます。これらのいずれか、もしくは全部のサイズをゼロとすることもできます。

色

パディングは、常に要素の背景色と同じ色になります。もし背景色を設定した場合には、その色が、その要素自身とパディングに対して適用されます。マージンは、常に透過です。

図表省略

要素をボーダーを使って線、枠で装飾することができます。要素の四方に同じボーダーを指定するためには、ボーダープロパティを使用します。幅（スクリーンに表示するため、通常はピクセルで）、スタイル、色を指定します。

図表省略

スタイルにnoneまたは、hiddenを設定し、明示的にボーダーを削除することもできます。また、色に透過を設定してレイアウトを変更せずにボーダーを非表示にすることができます。

上下左右のボーダーを個別に指定する場合には、個々のプロパティ（borer-top、border-right、border-bottom、border-left）を利用します。これらのプロパティを利用すると、上下左右のボーダーをひとつだけ指定したり、それぞれ違うボーダーを指定したりすることができます。

ボーダーの例

見出し要素の背景色と上ボーダーのルールセット: 

h3 {
  border-top: 4px solid #7c7; /* mid green */
  background-color: #efe;     /* pale green */
  color: #050;                /* dark green */
}

マージンとパディング

margin（要素外の余白幅） と padding（要素内の余白幅） を使うと、要素の位置や要素のまわりのスペースを調整することができます。margin、paddingを設定することで、要素外の余白幅、要素内の余白幅をそれぞれ設定することができます。

幅に数値をひとつ指定すると、その値は四方（上、右、下、左）に適用されます。数値をふたつ指定すると、ひとつ目の数値は上と下に、ふたつ目の数値は右と左に適用されます。上、右、下、左の順番で、四方すべての数値を別々に指定することもできます。

margin と padding の使用例

この例では、remarkクラスを使用して、段落を赤の枠線で囲んでいます。paddingの指定で、テキストと枠線の間にに4pxの幅が作られます。margin-leftの指定では、この段落要素の左側にある要素との間に、相対的に幅が作られます。

p.remark {
  border: 2px solid red;
  padding: 4px;
  margin-left: 24px;
}

さらに詳しく

marginやpaddingを使って要素のレイアウトを調整する場合、作成したスタイルルールはブラウザのデフォルトルールと複雑に作用しあいます。ブラウザが異なれば要素をどうレイアウトするかも異なります。スタイルシートで変更を加える前は、結果は似たような見た目かもしれませんが、スタイルシートで変更した箇所が思わぬ結果になることも時々あります。

期待どおりのレイアウト結果を得るためには、ドキュメントのマークアップを変更しなければならないこともあるかもしれません。これについては、チュートリアルの次のページで詳しく述べます。padding、 margin、 boderに関する更に詳しい情報は、Boxモデルのリファレンスページを参照してください。

課題: ボーダーを追加する

作成したCSSファイル style2.css を編集し、下記のルールを追加し、ページ内の各ヘディング要素に線を引いてください。

h3 {border-top: 1px solid gray;}

前のページの課題を実施した場合は、すでに作成したルールを修正するか、各リストアイテムの下に余白を追加するために下記の新規ルールを追加してください。

li {
  list-style: lower-roman;
  margin-bottom: 8px;
}
